# Emotion Images

Place emotion image files here to replace emojis in the Identify game.

Supported formats: PNG, JPG, JPEG, WEBP

Naming convention: `{emotion}.{ext}`

Example:
- `happy.png`
- `sad.jpg`
- `angry.webp`

Available emotions:
- happy
- sad
- angry
- frustrated
- surprised
- fear
- disgust
- neutral
