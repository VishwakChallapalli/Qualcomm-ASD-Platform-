# Qualcomm ASD Platform

---

## Information for running the code from Raunak

### Prerequisites
- Node.js installed (v18+ recommended)

### Run Front End
- Open a terminal in vscode
- cd into the folder: Qualcomm-ASD-Platform-
  - cd Qualcomm-ASD-Platform-
- run "npm install"
- run "npm run dev"

### Set Up Backend
- First if you haven't already, cd into server from Qualcomm-ASD-Platform- and then create a file called .env
- copy the contents of .env.example into .env and replace <username> and <password> respectively with the ones you have been provided

### Run Backend
- Open a another terminal
- From Qualcomm-ASD-Platform- cd into server
  - cd server
- run "npm install"
- run "node index.js"

Login and signup will not work without the back end running
You wll get errors if your mongo db credentials not correct.
